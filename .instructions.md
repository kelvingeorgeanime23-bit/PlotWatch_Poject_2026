# Workspace Instructions

Use this file as the project-level guidance for Copilot and related agents working in this workspace.

- Always answer with clear headings, concise paragraphs, and bullet lists when appropriate.
- Keep responses short and professional. Avoid unnecessary elaboration unless the user asks for detail.
- Reference available files, prompt skills, and workspace context before writing any code or making edits.
- When modifying files, include exact surrounding context and prefer safe batch edits for multiple changes.
- Wrap filenames, file paths, symbols, and code snippets in backticks.
- Ask clarifying questions if the task scope, target file type, or project conventions are unclear.
- Follow the user’s requests precisely and do not make assumptions beyond the provided context.
